What are Regular Expressions?

A regular expression is a pattern that you can use to search for and match a very specific string of text. A regular expression can search for any type of character, which could include any letter, number, symbol, and unicode letters for foreign languages. It also can pattern match whitespace and escaped or hidden characters like a new line or a tab. Here's a look at using regex to search for a name:

The following example is from Learn Regex the Hard Way.

name.match(/^[\da-z]+\d$/)

If you've never seen regex, this probably looks like gibberish. Let's take a closer look and break it down:

^ Match the beginnig of the string or line

[ ] The square brackets match a set of characters.

[\da-z] \d matches any digit from 0-9 and then a-z is matched

+ Match the preceding character set one or more times

\ Match any digits from 0-9

$ Match the end of the line (opposite of the ^ symbol)

So basically, this is searching for any string with any digit from 0-9 and any letter from a-z until the end of the line.

You can see that regex is a very powerful tool. You could use it to search for any kind of character pattern, identify it and manipulate it. This can be used to validate user input for e-mail addresses, phone numbers, usernames. It can also be used to rename hundreds of file names at once, advanced search and replace and editing file names. Regex is often used by localization engineers and there are hundres of reasons a a localization engineer may need to search and protect strings in a file from being translated.

Intro Reference to Character Matching

Characters
. dot: Match any character except line breaks Example: everything is matched
\w Word - Match any word character, same as [A-Za-z0-9]. Opposite is \W which matches anything that is not a word. Example: each word matched
\d \D Digit/Not a Digit - Lowercase d matches any digit 0-9 and uppercase matches any character not a digit. Example: +1-(444) 555-9302
\s \S Whitespace/Not whitespace - Not white space (\S) matches anything that is not a whitespace, including spaces, tabs and lines breaks. Example: +1-(444) 555-9302
